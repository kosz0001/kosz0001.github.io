    5 New Tags Learned

    <img>
    is a singular tag that inserts an image link as well as the alternate description for back-up and for accessibility.
    <a></a>
    is a paired tag that inserts a link a webpage, but this must be used with the 'href' attribute to denote a location for this tag to direct to.
    <nav></nav>
    is a paired tag which encompasses a set of links that navigate throughout the entire website, however it is just a block or a parent tag to encompass smaller tags within such as 'a href'.
    <table></table>
    is a paired tag which acts as a "container" in which a table, and all the contents of the table (the table caption, table header, table body etc.) reside within.
    <dl></dl>
    is a paired tag that is used to encompass a description list that denotes an item, this tag only denotes that this is a set and must include a <dt></dt> for the term to be described followed by <dd></dd>, which is the description of the paired term.
